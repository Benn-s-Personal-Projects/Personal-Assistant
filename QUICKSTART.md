# 🚀 QUICK START GUIDE

## Get Your Personal Assistant Running in 5 Minutes!

### Step 1: Install Expo Go (2 minutes)

📱 **On your phone:**
- **iPhone**: Open App Store → Search "Expo Go" → Install
- **Android**: Open Play Store → Search "Expo Go" → Install

### Step 2: Setup on Computer (1 minute)

Open Terminal/Command Prompt and run:

```bash
cd personal-assistant-mobile
npm install
```

### Step 3: Start the App (1 minute)

```bash
npm start
```

A QR code will appear!

### Step 4: Open on Phone (1 minute)

**iPhone:**
1. Open Camera app
2. Point at QR code
3. Tap the notification that appears
4. App opens in Expo Go!

**Android:**
1. Open Expo Go app
2. Tap "Scan QR Code"
3. Point at QR code
4. App loads automatically!

### Step 5: Start Chatting! 🎉

Try asking:
- "What's my latest utility bill?"
- "How much water did I use?"
- "What are my tasks?"
- "Show me my recent expenses"

## 💡 Pro Tips

- Keep the terminal window open while using the app
- Shake your phone to reload or access developer menu
- Your data is fetched directly from Supabase
- Claude AI analyzes your actual personal data

## ⚡ Super Fast Setup (Alternative)

If you're comfortable with terminal:

```bash
cd personal-assistant-mobile
./setup.sh
npm start
```

Then scan the QR code with Expo Go!

## 🆘 Need Help?

**App won't load?**
- Make sure phone and computer are on same WiFi
- Try restarting `npm start`

**Can't scan QR?**
- In terminal, press `t` to switch to tunnel mode
- Try scanning again

**Seeing errors?**
- Check that all dependencies installed: `npm install`
- Verify Node.js is installed: `node --version`
